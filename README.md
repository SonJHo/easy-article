# 📰 easy-article
사회·정치 뉴스를 핵심 중심으로 분석하여  
복잡한 기사를 쉽고 빠르게 이해할 수 있도록 돕는 뉴스 분석 서비스입니다.

## 💡 개발 배경

젊은 세대를 중심으로 정치·사회 뉴스에 대한 관심이 감소하고 있다고 느꼈습니다.  
그 원인으로는 다음과 같은 문제들이 있다고 판단했습니다.

- 사건 정보가 여러 기사로 파편화되어 전체 흐름 파악이 어려움
- 정치·사회 분야 특유의 생소한 용어로 인한 높은 이해 장벽
- 핵심 내용을 파악하기까지 불필요하게 많은 시간 소요

이러한 문제를 해결하기 위해  
복잡한 사회·정치 기사를 **핵심 위주로 요약**하고,  
기사에 등장하는 **어려운 용어를 문맥에 맞게 쉽게 설명해주는 뉴스 분석 서비스**를 기획·개발하게 되었습니다.

---

## 🎯 주요 기능

- 뉴스 기사 핵심 요약
- 기사 내 핵심 키워드 및 주요 이슈 추출
- 정치·사회 용어 자동 설명
- 네이버 뉴스 API를 활용한 기사 수집
- Redis 캐싱을 통한 응답 속도 개선
